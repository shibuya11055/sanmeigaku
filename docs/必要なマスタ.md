# マスタ

命式計算に必要なマスタはDBへ保存せず、コード内の静的データとして管理する。

## `Sanmeigaku::StaticData`

- 五行（Element）
- 十干（Stem）
- 十二支（Branch）と蔵干
- 60干支（Cycle）と天中殺
- 十大主星
- 十二大従星
- 六親法の原点データ

## ActiveHash

- `LunarCalendarEntry`: 1925〜2044年の月入日
- `YearlyDayNumber`: 年ごとの日番号基準
- `CycleMapping`: 60干支の互換マスタ
- `ElementRelation`: 五行関係
- `Prefecture` / `Plan` / `HeavenlyVoidWavePattern`

これらはアプリケーションコードから参照し、命式計算時にPostgreSQLへアクセスしない。
